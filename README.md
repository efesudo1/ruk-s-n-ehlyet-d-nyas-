# Rükiş'in Ehliyet Dünyası 🚗

Ehliyet sınavı hazırlığı için interaktif bir web uygulaması.

## 📋 Proje Hakkında

Bu uygulama, trafik işaretlerini öğrenmek ve ehliyet sınavına hazırlanmak için geliştirilmiş bir PWA (Progressive Web App) uygulamasıdır.

## ✨ Özellikler

- 📱 Mobil dostu responsive tasarım
- 🎨 Renkli ve kullanıcı dostu arayüz
- 🚩 Trafik işaretleri ve kuralları
- 💾 İnternet bağlantısı olmadan kullanılabilir (PWA)

## 📁 Dosya Yapısı

```
.
├── index.html          # Ana HTML dosyası
├── manifest.json       # PWA Manifest dosyası
├── download_signs.py   # Trafik işaretlerini indiren Python script
├── assets/             # Uygulamanın kaynakları
├── images/             # Görüntü dosyaları
└── README.md          # Bu dosya
```

## 🚀 Kullanım

1. `index.html` dosyasını bir web tarayıcısında açın
2. Veya GitHub Pages üzerinden erişin

## 🛠️ Geliştirme

Python scripti kullanarak trafik işaretlerini güncellemek için:

```bash
python download_signs.py
```
